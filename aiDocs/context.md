- Product direction: cross-platform mobile app with React Native (post-validation)
- Current build target: responsive web MVP demo (desktop + mobile browser)

- Deployment target for MVP: static hosting (Vercel, Netlify, or GitHub Pages)

## Current Focus

- Build and iterate on the web MVP flow: landing -> quiz -> results -> interactive demo -> waitlist
- Keep messaging privacy-first and clear: no selling/sharing user data
- Optimize for fast iteration and shareability with stakeholders/advisors
